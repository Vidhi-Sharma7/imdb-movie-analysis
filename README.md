# 🎬 IMDb Movie Analysis Dashboard

## 📊 Project Overview
This project analyzes IMDb movie data to uncover trends in ratings, genres, and voting patterns.  
The data was cleaned using Python and visualized using Power BI.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Power BI
- Data Cleaning & Visualization

---

## 📌 Key Insights
- Drama and Documentary are the most common genres
- Movie production has increased significantly over time
- Highly rated movies tend to have higher vote counts
- Top movies consistently maintain ratings above 9
